# [Features](FEATURES.md)

**`Pokemon Emerald XTREME`** offers hundreds of features from various [core series Pokémon games](https://bulbapedia.bulbagarden.net/wiki/Core_series), along with popular quality-of-life enhancements designed to streamline development and improve the player experience. A full list of those features can be found in [`FEATURES.md`](FEATURES.md).

# Compatibility with other roms

- **`Pokemon Emerald XTREME`** supports multiplayer functionality with other games built on **`pokeemerald-expansion`** and with national dex ranging from gen 1 to gen 7 pokemons.
